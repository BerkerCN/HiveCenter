# Security Policy

## Supported Versions

Currently, the `main` branch (`v0.2-beta`) and official releases are actively supported with security updates. 

| Version | Supported          |
| ------- | ------------------ |
| v0.2.x  | :white_check_mark: |
| v0.1.x  | :x:                |

## Reporting a Vulnerability

Security is a primary priority for HiveCenter, as it is an autonomous multi-agent system that can execute shell commands and modify files on the host.

If you discover a vulnerability, **do not open a public issue** with exploit details.

**Preferred:** use [GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) for this repository (**Security** tab → **Report a vulnerability**). That keeps the report private until a fix exists.

We will investigate legitimate reports and aim to acknowledge them within 48 hours.
